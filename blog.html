Here is my first blog entry, and rather ambitiously I'm writing it directly into the text editor, which is needless to say an interesting experience. The primary experience of interest being the complete understanding of how dependent on this text-editor I am, but isn't that the crux of most technology?

So I suppose I've been procastinating on these blog entries because I don't feel like setting up Jekyll, which I hope doesn't come across as lazy. I'd say the procastination stems from not knowing precisely which branch of programming to pursue. I don't really fancy myself a design person, though I seem drawn to using CSS and javascript to create static websites, even if I do so rather poorly. And I'm not trying to be modest here...if you like my design decisions, thank you, but the websites that really work, that make a difference in how one understands a product, person, idea through the internet, that's something special, and something I don't purport to be able to do. But it is super fun to try!
